# WindLand - Новая версия (PHP 7.4+ | UTF-8)

## 📋 Описание
Полностью переписанная браузерная ролевая игра WindLand с современной архитектурой и поддержкой UTF-8.

## ✨ Особенности
- **PHP 7.4+** - современный синтаксис и типизация
- **UTF-8 (utf8mb4)** - полная поддержка Unicode
- **MySQLi** - безопасное подключение к базе данных
- **Подготовленные выражения** - защита от SQL-инъекций
- **Адаптивный дизайн** - работает на всех устройствах

## 🚀 Установка

### 1. База данных
```bash
mysql -u root -p < database/schema.sql
```

### 2. Настройка конфигурации
Отредактируйте `configs/config.php`:
```php
define('SQL_HOST', 'localhost');
define('SQL_USER', 'ваш_пользователь');
define('SQL_PASS', 'ваш_пароль');
define('SQL_BASE', 'windland_db');
```

### 3. Права доступа
```bash
chmod 755 /path/to/WindLand_New
chmod 644 configs/config.php
```

### 4. Картинки
Скопируйте изображения из старой версии:
```bash
cp -r /workspace/WindLand/public_content/* /workspace/WindLand_New/public_content/
cp -r /workspace/WindLand/images/* /workspace/WindLand_New/images/
```
