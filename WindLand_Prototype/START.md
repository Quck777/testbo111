# 🎮 WindLand Prototype - Полностью готов к запуску!

## ✅ Создано:

**Файлы проекта:**
- `/workspace/WindLand_Prototype/index.php` - Единый рабочий файл игры (439 строк)
- `/workspace/WindLand_Prototype/README.md` - Полная документация
- `/workspace/WindLand_Prototype/public_content/` - Папки для картинок:
  - `avatars/` - аватарки персонажей
  - `locations/` - изображения локаций  
  - `items/` - предметы инвентаря

## 🚀 Как запустить:

### Шаг 1: Настройте подключение к БД
Откройте `index.php` и измените строки 9-12:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');      // Ваш пользователь MySQL
define('DB_PASS', '');          // Укажите ваш пароль MySQL!
define('DB_NAME', 'windland_proto');
```

### Шаг 2: Разместите на сервере
Скопируйте папку `WindLand_Prototype` в корень вашего веб-сервера:
- Apache: `/var/www/html/`
- XAMPP: `htdocs/`
- OpenServer: `domains/localhost/`

### Шаг 3: Откройте в браузере
Перейдите по адресу: `http://localhost/WindLand_Prototype/`

**База данных создастся автоматически!**

## 🎯 Что работает:

✅ **Регистрация** - создание нового аккаунта с валидацией
✅ **Вход/Выход** - авторизация через password_hash/password_verify
✅ **Главная страница** - отображение текущей локации с картинкой
✅ **Перемещение** - переходы между локациями (Город ↔ Лес/Пещера/Таверна)
✅ **Чат** - отправка и просмотр сообщений в чате локации
✅ **Профиль** - характеристики персонажа (HP, золото, опыт, уровень)
✅ **Инвентарь** - просмотр предметов с иконками
✅ **Навигация** - боковое меню с быстрым доступом

## 🖼️ Пути к картинкам (как в оригинале):

```html
<!-- Аватарка персонажа -->
<img src="public_content/avatars/no_avatar.png">

<!-- Изображение локации -->
<img src="public_content/locations/city_center.png">

<!-- Предмет в инвентаре -->
<img src="public_content/items/sword_rust.png">
```

Если картинка не найдена - автоматически подставляется заглушка.

## 🗄️ База данных:

Автоматически создаются 5 таблиц при первом запуске:

1. **users** - пользователи (id, login, password, email, level, exp, gold, hp, location_id...)
2. **locations** - локации (id, name, description, image, parent_id)
3. **items** - предметы (id, name, type, price, image)
4. **user_items** - инвентарь (id, user_id, item_id, count)
5. **chat** - сообщения (id, user_id, message, date)

Тестовые данные добавляются автоматически:
- 4 локации (Город, Лес, Пещера, Таверна)
- 3 предмета (Меч, Зелье, Броня)

## 📝 Примеры использования:

### Зарегистрироваться:
1. Откройте главную страницу
2. Нажмите "Регистрация"
3. Введите логин (минимум 3 символа), email, пароль
4. Нажмите "Создать персонажа"

### Войти:
1. Введите логин и пароль на главной
2. Нажмите "Войти"

### Перемещаться:
- На главной странице нажмите кнопки переходов (⬅️ Город, ➡️ Лес и т.д.)

### Общаться в чате:
- Внизу страницы главной введите сообщение и нажмите Send

### Смотреть инвентарь:
- В боковом меню нажмите "🎒 Инвентарь"

## 🛠️ Добавление контента:

### Добавить новую локацию через SQL:
```sql
INSERT INTO locations (name, description, image, parent_id) 
VALUES ('Замок', 'Древний замок короля', 'castle.png', 1);
```

### Положить картинку:
Сохраните файл `castle.png` в папку `public_content/locations/`

### Выдать предмет игроку:
```sql
INSERT INTO user_items (user_id, item_id, count) VALUES (1, 1, 1);
```
