# WindLand Prototype - Полностью рабочая версия

## 🚀 Быстрый старт

### 1. Требования
- PHP 7.4+ или выше
- MySQL 5.7+ или MariaDB
- Веб-сервер (Apache/Nginx)

### 2. Установка

#### Вариант A: Автоматическая установка БД
Просто откройте `index.php` в браузере - база данных создастся автоматически!

#### Вариант B: Ручная настройка
1. Откройте `index.php` и укажите ваши данные БД:
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', 'ваш_пароль'); // Укажите пароль MySQL
define('DB_NAME', 'windland_proto');
```

2. База данных и таблицы создадутся автоматически при первом запуске

### 3. Запуск
Откройте в браузере: `http://localhost/WindLand_Prototype/`

## 🎮 Функционал

✅ **Регистрация** - создание нового персонажа
✅ **Вход/Выход** - авторизация в игре
✅ **Профиль** - просмотр характеристик персонажа
✅ **Локации** - перемещение между локациями (Город, Лес, Пещера, Таверна)
✅ **Чат** - общение в чате локации
✅ **Инвентарь** - просмотр предметов
✅ **Система HP** - полоска здоровья
✅ **Уровни и опыт** - система прогрессии
✅ **Золото** - игровая валюта

## 📁 Структура проекта

```
WindLand_Prototype/
├── index.php              # Единый файл игры (все функции)
├── public_content/        # Картинки
│   ├── avatars/          # Аватарки персонажей
│   ├── locations/        # Изображения локаций
│   └── items/            # Предметы инвентаря
└── README.md             # Этот файл
```

## 🖼️ Пути к картинкам

Все пути сохранены как в оригинальной игре:

- Аватарки: `public_content/avatars/{avatar_name}`
- Локации: `public_content/locations/{location_image}`
- Предметы: `public_content/items/{item_image}`

По умолчанию используются заглушки если картинка не найдена:
- `no_avatar.png` - аватар по умолчанию
- `loc_default.png` - локация по умолчанию
- `item_default.png` - предмет по умолчанию

## 🗄️ Структура БД

Автоматически создаются 5 таблиц:

1. **users** - пользователи (логин, пароль, характеристики)
2. **locations** - локации (название, описание, переходы)
3. **items** - предметы (название, тип, цена)
4. **user_items** - инвентарь пользователей
5. **chat** - сообщения чата

При первом запуске создаются:

**Локации:**
- Город (центральная)
- Лес (доступен из Города)
- Пещера (доступна из Города)
- Таверна (доступна из Города)

**Предметы:**
- Ржавый меч (оружие, 10 золотых)
- Зелье лечения (зелье, 5 золотых)
- Кожаная броня (броня, 20 золотых)

## 💡 Как добавить контент

### Добавить новую локацию:
```sql
INSERT INTO locations (name, description, image, parent_id) 
VALUES ('Новая локация', 'Описание', 'image.png', 1);
```

### Добавить предмет:
```sql
INSERT INTO items (name, type, price, image) 
VALUES ('Меч героя', 'weapon', 100, 'sword_hero.png');
```

### Выдать предмет игроку:
```sql
INSERT INTO user_items (user_id, item_id, count) 
VALUES (1, 1, 1); -- user_id=1, item_id=1
```

## 🛠️ Расширение

Для добавления новых функций редактируйте `index.php`:

- Новые страницы: добавьте `elseif ($page == 'newpage')` блок
- Новые действия: добавьте обработку в `$action` switch
- Новые таблицы: добавьте CREATE TABLE в схему
